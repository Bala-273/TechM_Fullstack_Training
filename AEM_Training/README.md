# AEM Training 

This repository contains the code for the daily tasks given during the AEM Training and also for the final project, where a news website is built using Adobe Experience Manager (AEM).

## Codebase Directory

All the code for the daily tasks and the final project can be accessed from the **`codebase`** directory. 

### Task Structure:
- A separate folder is created for each **daily task**, with the folder name corresponding to the **date** the task was given.
- The entire tasks including the final project share a common codebase directory.
- Also the code for each task is provided in the corresponding folder in the **`README.md`** file format.
- The sample output is added for each and every task in a separate folder
- For final project i.e. building a news website, the output screenshots are in the FinalProject_Output folder

## Accessing the Code

1. Navigate to the **`codebase`** directory to access code for all the tasks and final project.
2. You can also access the **`README.md`** file in each folder to view the task details and code for that particular day.
